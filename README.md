# Détecteur de Fake News - Projet 7
## Par Youness Bourek - M1 Microélectronique

### Description
Application web de détection de fausses informations utilisant la régression logistique et TF-IDF.

### Installation
```bash
pip install -r requirements.txt
python app.py
```

### Accès
Ouvrir le navigateur à l'adresse : http://localhost:5000

### Fonctionnalités
- Analyse de texte en temps réel
- Indication de confiance
- Exemples prédéfinis
- Interface responsive

### Technologies
- Python / Flask
- Scikit-learn (Régression Logistique)
- NLTK (Traitement NLP)
- HTML / CSS / JavaScript
